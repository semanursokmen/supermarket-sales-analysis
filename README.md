# Supermarket Sales Analysis

## Project Overview
This project analyzes supermarket sales data to identify trends across product categories, cities, and customer gender using SQL, Python, and Excel.

## Objectives
- Analyze sales performance across product lines  
- Compare sales by city  
- Evaluate sales differences by gender  
- Extract insights using SQL queries  
- Visualize data using Python and Excel  

## Dataset
The dataset contains over 1,000 sales records, including:
- Invoice ID  
- Branch and City  
- Customer Type and Gender  
- Product Line  
- Unit Price, Quantity, and Total Sales  
- Date, Time, and Payment Method  

## Tools Used
- SQL (SQLite)  
- Python (Pandas, Matplotlib, Seaborn)  
- Microsoft Excel (Pivot Tables & Charts)  

## Process
- Data stored and queried using SQLite database  
- SQL is used for data extraction and aggregation  
- Python is used for data analysis and visualization  
- Excel is used for additional validation and summary insights  

## Visualizations
- Sales by Product Line (Bar Chart)  
- Sales by City (Bar Chart)  
- Sales by Gender (Bar Chart)  

## Key Insights
- Food and Beverages is the highest-performing product category  
- Sales vary across different cities  
- Female customers contribute slightly higher total sales compared to male customers  

## SQL Analysis
SQL queries were used to analyze:
- Total sales by product line  
- Sales by city  
- Sales by gender  
- Identification of top-performing categories

This analysis provides insights into customer behavior and sales distribution, helping to better understand business performance and support data-driven decision-making.
