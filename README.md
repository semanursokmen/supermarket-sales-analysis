# Supermarket Sales Analysis

## 📊 Project Overview
This project analyzes supermarket sales data to identify trends across product categories, cities, and customer gender using Excel pivot tables and visualizations.

## 🎯 Objectives
- Analyze sales performance across product lines  
- Compare sales by city  
- Evaluate sales differences by gender  
- Visualize insights using charts  

## 🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts (Bar & Pie Charts)  

## 📈 Key Insights
- Health and Beauty is one of the top-performing product categories  
- Naypyitaw shows strong sales performance compared to other cities  
- Female customers contribute slightly higher to total sales  

## 📊 Visualizations
- Sales by Product Line (Bar Chart)  
- Sales by City (Pie Chart)  
- Sales by Gender (Bar Chart)  

## 📁 Dataset
The dataset contains 1,000+ sales records including:
- Invoice ID  
- Branch and City  
- Customer Type and Gender  
- Product Line  
- Unit Price, Quantity, and Total Sales  
- Date, Time, and Payment Method  

## 🚀 Conclusion
This analysis provides insights into customer behavior and sales distribution, which can help improve business decisions and marketing strategies.
