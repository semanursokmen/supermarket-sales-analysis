# Supermarket Sales Analysis

## Project Overview
This project analyzes supermarket sales data to identify trends across product categories, cities, and customer gender using Excel pivot tables, visualizations, and SQL queries.

## Objectives
- Analyze sales performance across product lines  
- Compare sales by city  
- Evaluate sales differences by gender  
- Extract insights using SQL queries  
- Visualize data using charts
  
## Dataset
The dataset contains 1,000+ sales records, including:
- Invoice ID  
- Branch and City  
- Customer Type and Gender  
- Product Line  
- Unit Price, Quantity, and Total Sales  
- Date, Time, and Payment Method
  
## Tools Used
- Microsoft Excel  
- SQL (SQLite)  
- Pivot Tables  
- Charts (Bar & Pie Charts)  

## Visualizations
- Sales by Product Line (Bar Chart)  
- Sales by City (Pie Chart)  
- Sales by Gender (Bar Chart)
  
## Key Insights
- Health and Beauty is one of the top-performing product categories  
- Naypyitaw shows a strong sales performance compared to other cities  
- Female customers contribute slightly higher to total sales  

## SQL Analysis
SQL was used to query and analyze the dataset:

- Total sales by product line  
- Sales by city  
- Sales by gender  
- Identification of top-performing categories  

## Conclusion
This analysis provides insights into customer behavior and sales distribution, helping to better understand business performance and support data-driven decision-making.
